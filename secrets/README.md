# Secrets

Este diretório armazena modelos de variáveis de ambiente para segredos.

## Uso

1. Copie `secrets/vtex.env.example` para `secrets/vtex.env`
2. Insira seu valor real em `VTEX_APP_TOKEN`
3. Não comite `secrets/vtex.env` no Git

## Arquivo seguro

O arquivo real deve permanecer local e protegido. Use o exemplo apenas como modelo.
